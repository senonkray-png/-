# Denis Milev — AI Solutions Architect Portfolio

A premium one-page portfolio website built with **Next.js 15**, **TypeScript**, **Tailwind CSS**, **Framer Motion**, and **Lenis** smooth scroll.

## Design
- **Style:** Dark Luxury Minimalism + Heavy Glassmorphism + Neon Purple
- **Background:** `#0a0a0a` (deep black)
- **Accent:** `#a855f7` / `#c084fc` (neon purple)

## Sections
1. **Hero** — Name, typing animation, neon ring avatar, parallax scroll
2. **About** — Glassmorphism card with profile text and stats
3. **Competencies** — 6-category skill grid with icons
4. **Timeline** — Animated vertical experience timeline with glowing line
5. **Contact** — Social links + main CTA

## Setup

```bash
npm install
npm run dev       # development server at http://localhost:3000
npm run build     # production build
npm run start     # production server
```

## Customization

Edit `src/data/profile.ts` to update all content (name, bio, skills, experience, contacts).

Replace `public/resume.pdf` with your actual PDF resume.
